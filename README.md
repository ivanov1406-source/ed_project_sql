## ed_project_sql overview  

Компания планирует конференцию в Бостоне в первой половине 2017 года.
Для участников необходимо организовать проживание через Airbnb.

---

## Условия задачи

- Всего участников: **150 человек**
- Продолжительность проживания: **3 дня**
- Место проведения: **Boston**
- Тип размещения: **квартиры Airbnb**

### Особые условия

- **10 топ-менеджеров** должны проживать **в отдельных квартирах**
- Остальные участники могут проживать **совместно**
- Максимальный бюджет на проживание: **$70,000**
- Минимальная оценка жилья участниками: **8/10**

---

## Задачи анализа
1. Найти **оптимальные даты конференции** с точки зрения стоимости проживания и уровня удовлетворенности.
2. Подобрать **набор квартир**, позволяющий разместить **150 участников**.

---

## Данные

Датасет:  
https://www.kaggle.com/datasets/airbnb/boston

Используемые таблицы:
- `listings`
- `calendar`

### Folder Description

**data/raw/**  
Исходные данные из датасета Airbnb Boston.  
https://www.kaggle.com/datasets/airbnb/boston

**data/processed/**  
Очищенные и подготовленные таблицы, используемые для анализа.

**sql/**  
SQL-запросы для очистки данных, формирования рейтингов и поиска оптимальных дат и квартир.
