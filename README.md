# ed_project_sql
1. Условия задачи
Компания планирует конференцию в Бостоне в первой половине 2017 года.
Необходимо разместить 150 участников на 3 дня в квартирах Airbnb.
Особые условия:
- 10 топ-менеджеров должны жить отдельно
- остальные могут жить совместно
- бюджет проживания $70.000, но лучше потратить меньше.
- оценка жилья участниками на 8/10 или выше.

2. Необходимо:
	1. Найти оптимальные даты конференции по стоимости проживания и уровню удовлетворенности.
	2. Подобрать список квартир для размещения 150 участников.

3. Данные
https://www.kaggle.com/datasets/airbnb/boston

4. Основные результаты
Рекомендуемые даты конференции: 20–22 февраля 2017
Цена размещения топ-менеджера: $610
		участника: $399
Общий бюджет: $62 005
Ожидаемый уровень удовлетворенности жильём: 9.4
Для размещения выбрано:
- 10 отдельных квартир для топ-менеджеров
- 57 квартир для остальных участников
